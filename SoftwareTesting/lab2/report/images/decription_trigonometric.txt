Для тригонометрического неравенства было реализовано три модуля:
1. TrigonometricBase
2. TrigonometricFunctions
3. TrigonometricExpression
В первом модуле осуществляется подсчет функции sin(x) с помощью разложения в ряд. Во втором реализованы основные тригонометрические функции, 
выраженные через функцию sin первого модуля. В третьем же вычисляется конечное неравенство системы.
Для тестирования необходимо реализовать полное покрытие, то есть выбрать классы эквивалентности, где функция изменяется подобным образом:
сalcSin. Классы эквивалентности:
- sin(x)>0 при x принадлежащем (2Pin; Pi+2Pin), n принадлежит z.
- sin(x)<0 при x принадлежащем (-Pi+2Pin;2Pin), n принадлежит z.
- Промежутки возрастания. x принадлежит (-Pi+2Pin;Pi/2+2Pin), n принадлежит z.
- Промежутки убывания. x принадлежит (-Pi/2+2Pin;3Pi/2+2Pin), n принадлежит z.
сalcCos. Классы эквивалентности:
- cos(x)>0 при x принадлежащем (-Pi/2+2Pin; Pi/2+2Pin), n принадлежит z.
- cos(x)<0 при x принадлежащем (Pi/2+2Pin;3Pi/2+2Pin), n принадлежит z.
- Промежутки возрастания. x принадлежит (-Pi+2Pin;2Pin), n принадлежит z.
- Промежутки убывания. x принадлежит (2Pin;Pi+2Pin), n принадлежит z.
сalcTan. Классы эквивалентности:
- (-Pi/2+2Pin; Pi/2+2Pin), n принадлежит z.
- Точки разрыва Pi/2+2Pin, n принадлежит z.
сalcCot. Классы эквивалентности:
- (2Pin; Pi+2Pin), n принадлежит z.
- Точки разрыва Pi+2Pin, n принадлежит z.
сalcSec. Классы эквивалентности:
- (-Pi/2+2Pin; Pi/2+2Pin), n принадлежит z.
- (Pi/2+2Pin;3Pi/2+2Pin), n принадлежит z.
- Точки разрыва Pi+2Pin, n принадлежит z.

Для конечного неравенства системы были выделены следующие классы эквивалентности:
- (-Pi/2+2Pin;Pi/2+2Pin), n принадлежит z.
- (Pi/2+2Pin;3Pi/2+2Pin), n принадлежит z.
- Устранимые точки разрыва в Pi+Pn, где n принадлежит Z.
- Разрывы второго рода в Pi/2 + Pn
